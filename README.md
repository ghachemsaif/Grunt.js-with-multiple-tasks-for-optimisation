# Grunt.js-with-multiple-tasks-for-optimisation
Setting up a gruntfile.js for your project 


Steps : 

1- Download all files 
2 - Load npm packages under /lib
3- Install Grunt 
4- Run Grunt
5- two folder will be created ( /dev and /prod ) 
/prod : it's a an optimisation version 
/dev : it's the folder wich containes all your assests for the dev environnement 



All html files under /dev and /prod will be created automaticly after setting the commande grunt, Only html files in the root should be modified
